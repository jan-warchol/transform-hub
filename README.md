IFCA implementation in Python
=============================

Environment setup
-----------------

_Tested with Python 3.8.10 and Ubuntu 20.04._

Check Python version:

    $ python3 --version
    Python 3.8.10


Usage
-----

Run test cases:

    python3 test-cases.py

Run with debug output and 10x faster:

    PYFCA_DEBUG=1 python3 test-cases.py 0.1
